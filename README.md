# 🎓 Student Attendance Management System (Django)

A Django-based web application for managing students and tracking daily attendance in an academic environment.

---

## 🚀 Features

- Add, edit, and delete students
- Mark attendance (Present / Absent)
- Track attendance by date
- View student attendance history
- Calculate attendance percentage
- Simple and clean dashboard

---

## 🧱 Tech Stack

- Python 3
- Django Framework
- SQLite Database
- HTML, CSS (Django Templates)

---

## 📊 System Overview

This system helps teachers efficiently track student attendance and generate reports.

Main components:
- Student Management
- Attendance Sessions
- Attendance Records
